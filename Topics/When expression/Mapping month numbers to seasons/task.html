Write a program that takes an integer as input representing a month number (1-12) and prints the corresponding season using a when expression. Months 12, 1, 2 are winter; 3, 4, 5 are spring; 6, 7, 8 are summer; and 9, 10, 11 are fall.


Sample Input:
3

Sample Output:
Spring


Sample Input:
7

Sample Output:
Summer



Memory limit: 256 MB
Time limit: 8 seconds
